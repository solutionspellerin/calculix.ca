# Calculix.ca

Calculateurs financiers gratuits (pourboires, investissement et plus).

Site trilingue FR / EN / ES — même architecture que [CalculDates.ca](https://www.calculdates.ca).

## Pages
- `index.html` — Accueil (cartes des outils)
- `pourboires.html` — Séparation de pourboires (équitable ou selon les heures travaillées)
- `investissement.html` — Projection d'investissement (capital initial + cotisation récurrente + rendement)
- `about.html` — À propos & divulgation affiliés
- `privacy-policy.html` — Politique de confidentialité (Loi 25, LPRPDE)
